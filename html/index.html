get /capabilities
capabilities api entry point, list the resources provided by the current API end-point. (capabilities)

Return type

capabilities

Example data

Content-Type: application/json
[ {
  "action" : "capabilities",
  "version" : 0.1
}, {
  "action" : "collections",
  "version" : 0.1
}, {
  "action" : "products",
  "version" : 0.1
}, {
  "action" : "routes",
  "version" : 0.1
} ]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Successful request capabilities

404

capabilities not available

500

Internal server error
